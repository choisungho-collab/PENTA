PENTA 녹화기

리그 오브 레전드를 켜두면 알아서 녹화하고, 게임이 끝나면 Riot 공식 기록으로 전적을 분석해 아카이브에 올립니다. 설치 과정 없이 압축만 풀면 바로 실행됩니다.

녹화기 받기 (.zip) Windows 10 / 11 · 무료 · 압축 풀고 실행

무엇을 하나요

자동 녹화

게임이 시작되면 감지해서 녹화를 켜고, 끝나면 멈춥니다. 따로 누를 게 없습니다.

Riot 기록 분석

KDA·CS·골드·아이템·룬과 골드 추이, 오브젝트까지 공식 데이터로 정리됩니다.

멀티 시점

같은 게임을 여럿이 녹화하면 자동으로 묶여, 한 교전을 여러 자리에서 다시 봅니다.

시작하기 4단계

  1. 받은 penta_recorder.zip의 압축을 풉니다. 생긴 penta_recorder 폴더 안의 _internal 폴더는 지우지 마세요.
  2. 설정 파일을 같은 폴더에 넣습니다. penta_recorder.exe 옆에 config.json 파일을 만들고 아래 내용을 채웁니다(다음 항목 참고). 프록시 주소와 Supabase 정보가 있어야 분석·업로드가 연결됩니다.
  3. penta_recorder.exe를 더블클릭합니다. 처음 한 번은 영상 도구(ffmpeg)를 자동으로 받아오니 1~2분 기다리면 됩니다.
  4. 리그 오브 레전드를 플레이하세요. 게임이 끝나면 영상과 전적이 아카이브에 자동으로 올라갑니다. 창은 켜둔 채로 두면 됩니다.

설정 파일 config.json

{
  "proxy_url": "https://mypenta.netlify.app",   // Riot 프록시(Netlify) 주소
  "platform": "kr",
  "supabase": {
    "url": "https://bsrvmesrygbfeqicquvq.supabase.co",
    "anon_key": "eyJ… (공개 anon 키)",
    "service_key": "eyJ… (이 PC에만 보관)",
    "bucket": "media"
  },
  "encoder": "auto",   // auto | nvenc | x264
  "scale": "auto",     // auto | source | 1080 | 720 | 480
  "fps": 30,
  "min_game_sec": 300    // 이보다 짧은 게임(리메이크 등)은 건너뜀
}
proxy_url
Netlify에 올린 Riot 프록시 주소. 이 주소를 통해서만 Riot API를 부르므로 키가 노출되지 않습니다.
supabase.url / anon_key
아카이브 데이터베이스 주소와 공개 읽기 키. 공개되어도 안전합니다(권한이 막혀 있음).
supabase.service_key
업로드 권한 키 — 이 PC에만 두고 절대 공유하지 마세요. 웹이나 깃허브에 올리면 안 됩니다.
encoder / scale / fps
auto면 알아서 최선으로 잡습니다. 녹화기 창의 옵션에서 바꿔도 되고, 바꾸면 다음 녹화부터 적용됩니다.

권장 사양

운영체제
Windows 10 / 11 (64-bit)
그래픽카드
NVIDIA 권장 — 하드웨어 인코딩(NVENC)으로 게임 끊김 없이 녹화됩니다. 내장 그래픽·AMD에서도 동작하지만 CPU 부담이 커집니다.
그래픽 드라이버
최신 버전 권장 — 오래된 드라이버에선 녹화가 시작되지 않을 수 있습니다.
메모리 / 저장공간
8 GB 이상 · 영상 임시 저장용 수 GB 여유(업로드 후 자동 정리)
인터넷
업로드 + 첫 실행 시 도구(ffmpeg) 다운로드에 필요

그래픽 드라이버

NVIDIA 드라이버를 최신으로

가장 자주 막히는 부분입니다. 드라이버가 오래됐으면 녹화가 시작되지 않습니다. NVIDIA 앱을 설치하면 그래픽카드를 자동으로 인식해 한 번에 최신으로 깔아줍니다.

NVIDIA 앱으로 설치 드라이버 직접 검색

처음 실행할 때

압축 풀기 전 "차단 해제" — 내려받은 penta_recorder.zip을 우클릭 → 속성 → 아래쪽 차단 해제에 체크하고 적용한 뒤 압축을 풀면 경고가 줄어듭니다.

"확인되지 않은 앱" 경고(SmartScreen) — 서명되지 않은 개인 제작 프로그램이라 Windows가 한 번 경고를 띄울 수 있습니다. 추가 정보 → 실행으로 진행하면 됩니다.

"스마트 앱 컨트롤"이 아예 차단할 때 — 실행 옵션조차 없이 막히면 스마트 앱 컨트롤(SAC)이 켜진 PC입니다. 설정 → 개인정보 및 보안 → Windows 보안 → 앱 및 브라우저 컨트롤 → 스마트 앱 컨트롤 설정에서 끌 수 있습니다. (한 번 끄면 다시 켜려면 Windows 재설치가 필요하니 신중히 결정하세요.) 대부분의 PC는 SAC가 꺼져 있어 이 화면을 보지 않습니다.

백신이 막을 때 — 드물게 강하게 막으면 예외(허용) 목록에 penta_recorder.exe를 등록해 주세요.
