사용 매뉴얼

녹화기를 켜두고 리그 오브 레전드를 플레이하면 끝입니다. 화면이 무엇을 뜻하는지, 설정은 어떻게 바꾸는지, 막힐 때는 어떻게 하는지 안내합니다.

녹화기 화면 한눈에

Ready / Recording
지금 상태. Ready는 게임을 기다리는 중, Recording은 녹화 중입니다. 게임을 켜면 자동으로 Recording으로 바뀝니다.
Capture · Encoder · Quality
지금 쓰는 캡처 엔진과 화질. 녹화 중에는 실제 값(예: Capture · WGC, Encoder · NVENC), 대기 중에는 설정값(Auto)이 표시됩니다.
☁ Cloud
아카이브 연결 상태. Cloud면 업로드 준비 완료, Key needed면 service_key가 빠진 상태입니다.
N games
아카이브에 올라간 경기 수입니다.
Gallery / Open folder
Gallery는 웹 아카이브 열기, Open folder는 녹화 폴더 열기. 아래 Settings에서 화질·엔진을, Log에서 진행 상황을 볼 수 있습니다.

녹화 흐름 4단계

  1. 녹화기를 켜둡니다. 한 번 실행해 두면 떠 있다가 게임을 감지합니다. 창은 닫지 말고 켜둔 채로 두세요.
  2. 리그 오브 레전드를 플레이합니다. 게임이 시작되면 자동으로 녹화가 켜지고, 화면이 Recording으로 바뀝니다.
  3. 게임이 끝나면 멈춥니다. 녹화가 자동으로 종료되고, 잠시 뒤 Riot 공식 기록으로 전적이 분석됩니다.
  4. 아카이브에 자동 업로드. 영상과 전적이 웹 갤러리에 올라갑니다. Gallery 버튼으로 바로 확인하세요.

녹화 설정 Settings

기본값(Auto)이 대부분 최선입니다. 바꾸면 다음 녹화부터 적용됩니다.

Quality
Auto · Source · 1080p · 720p · 480p. Auto는 최상 화질. 저장 공간이나 업로드가 부담되면 720p로 낮추세요.
Encoder
Auto · GPU(NVENC) · CPU(x264). Auto는 NVIDIA가 있으면 GPU로 녹화해 게임이 안 끊깁니다. GPU가 없으면 자동으로 CPU로 넘어갑니다.
Capture
Auto · WGC · DXGI · GDI. Auto면 알아서 잡습니다. 화면이 검게 녹화되면 WGC로 바꿔보세요. 전체화면도 잘 잡힙니다.
Monitor
Auto · Monitor 1 / 2 / 3. 모니터가 여러 대일 때 게임이 뜨는 화면을 고릅니다. Auto가 보통 맞게 잡습니다.

자주 묻는 질문

녹화가 안 켜져요

대부분 그래픽 드라이버가 오래된 경우입니다. NVIDIA 드라이버를 최신으로 올려주세요. 그래도 안 되면 Settings의 Capture를 WGC로 바꿔보세요.

영상이 아카이브에 안 올라가요

화면 우상단이 ☁ Cloud인지 확인하세요. Key needed로 떠 있으면 config.json에 service_key가 빠진 것입니다. 인터넷 연결도 함께 확인해 주세요.

경기 수가 이상해요

config.json의 supabase.url이 본인 아카이브 주소(bsrvm…)인지 확인하세요. 비어 있으면 기본값으로 채워집니다.

화면이 검게 녹화돼요

Settings → Capture를 WGC로 바꾸면 대부분 해결됩니다. 전체화면 게임도 잘 잡힙니다.

처음 켰더니 한참 멈춰 있어요

최초 1회는 영상 도구(ffmpeg)를 자동으로 받아옵니다. 1~3분 기다리면 Ready로 바뀝니다.

아직 안 받으셨다면

녹화기 내려받기

설치 과정 없이 압축만 풀면 바로 실행됩니다. 설치 안내와 권장 사양은 다운로드 페이지에 있습니다.
